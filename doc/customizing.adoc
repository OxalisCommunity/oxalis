= Customizing Oxalis [[customizing]]

== Plugin [[plugin]]

[source,xml,subs="verbatim,attributes"]
.Maven depencency
----
<dependency>
    <groupId>network.oxalis</groupId>
    <artifactId>oxalis-api</artifactId>
    <version>{techversion}</version>
    <scope>provided</scope>
</dependency>
----
